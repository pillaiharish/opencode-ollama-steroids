# __PROMPT_ID__

Local session folder for `__PROJECT_SLUG__`.

This folder is ignored by default. Keep raw prompts, logs, screenshots, and redacted drafts local unless a human explicitly chooses to publish sanitized summary material.

## Expected Files

- `PROMPT_CODER.md`: builder input prompt.
- `PROMPT_REVIEWER.md`: reviewer input prompt.
- `__PROMPT_ID_UPPER___PLAN.md`: builder plan.
- `__PROMPT_ID_UPPER___PLAN_REVIEW.md`: reviewer plan decision.
- `__PROMPT_ID_UPPER___IMPLEMENTATION.md`: builder implementation notes.
- `__PROMPT_ID_UPPER___TEST_RESULTS.md`: builder validation evidence.
- `__PROMPT_ID_UPPER___IMPLEMENTATION_REVIEW.md`: reviewer implementation decision.
- `__PROMPT_ID_UPPER___FIXES.md`: builder fix notes, only when needed.
- `__PROMPT_ID_UPPER___SIGNOFF.md`: reviewer final decision.
- `CURRENT_STATE.md`: current prompt status.

Raw logs and event streams must remain local and ignored.
